The Second Task : Practice Session

You are about to start the second task, but first you will be introduced to the task in the practice session. We will ask several questions to test your comprehension of this task, you have to be able to answer all these questions correctly before you can advance to the actual task session.

In the Second Task there are two participants are interacting in a pair, one participant is the sender and the other participant is the receiver. Both the sender and the receiver share a fixed amount of lab token worth {{ Constants.endowment }}. The Sender will decide how much the sender will retain. Then the rest of the lab token will be transferred to the receiver.
